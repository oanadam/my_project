
# Beauty News Project

This project contains articles and reports about beauty, created to showcase my research, writing, and analytical skills in the beauty industry.

## Included Files

- `beauty_news_report.txt` – A detailed report on recent beauty trends.
- `beauty_news_today.txt` – An informative article on the latest beauty news.
- `Vogue/` – Folder with examples of structured content for beauty publications.
- `import requests.py` – Python script used for data collection (optional, demonstrating technical skills).

## Description

This project demonstrates my ability to:

- Research and analyze trends in the beauty industry;
- Write clear, engaging, and well-documented articles;
- Organize information in a professional and accessible manner.

This project is part of my portfolio for the **Beauty Editor position at Condé Nast / Vogue**.
